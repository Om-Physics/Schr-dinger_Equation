# Schrödinger Equation Solutions

Repository for exploring numerical solutions to the Schrödinger Equation.
Includes methods such as Finite Difference, Runge-Kutta, and Crank-Nicolson,
with Python scripts and Jupyter notebooks for simulations and plotting.

## Directory Structure
- `notebooks/`: Interactive notebooks with code and visualization
- `scripts/`: Standalone scripts for numerical methods
- `data/`: Output data from simulations
- `plots/`: Wavefunction and energy level plots
- `docs/`: Theory notes and references
